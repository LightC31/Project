# Адаптивний веб-проєкт на Bootstrap

Цей проєкт є навчальним прикладом створення адаптивного сайту на Bootstrap.  
Макет складається з трьох колонок (LEFT, MAIN, RIGHT), які автоматично змінюють свою структуру залежно від ширини екрана:  
- при великій ширині відображаються **три колонки**,  
- при середній — **дві**,  
- при вузькій — **одна**.

## Функціонал і складові
- **HEADER** — дворівневе меню (основне + вкладене).  
- **LEFT** — слайдер з вертикальними зображеннями (не менше 4).  
- **MAIN** — основний контент, реалізований через компонент **акордеон**.  
- **Форма зворотного зв’язку** з мінімум 5 полями та кнопками.  
- **RIGHT** — додатковий контент (довільний).  
- **FOOTER** — текстовий блок із довільним наповненням.  
- Стилізація виконана у файлі **style.css**.
